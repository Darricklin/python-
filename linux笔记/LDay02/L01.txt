一、操作系统历史
1946世界上产生第一台计算机。
人机矛盾：1、只能一人使用 2、手工输入，速度慢
1965年：麻省理工，贝尔实验室，ATT设计开发一套操作系统。
1969年：进度缓慢，计划失败。
1970年：肯.汤姆逊移植了一个软件，定制了一套使用该软件的标准，Unix系统。
1970年推出Unix系统，将1970年作为计算机的元年。
1985年：windows推出。
Unix是传统美国人思想，按日、月年收费。
1991年：林纳斯.托瓦兹仿照Unix写了一套Linux系统。
Linux纯开源，不收费。
当时开发人员100人左右，核心开发人员5人左右。
1993年：使用Linux系统人员10W人

二、安装Linux真机
1、需要iso镜像文件
2、使用U盘安装，使用刻录软件镜iso刻录到U盘中
3、将U盘插入电脑，重启电脑，重启时按 f12/esc/del,选择U盘启动
4、安装时，选择自动分区，将win与Linux并存
5、注意：安装双系统时，需要有一个空盘，空盘颜色一般为黑色或绿色。如果没有空盘，可以使用删除卷操作，设置空盘。

十、常用命令  
1、cd  打开并进入一个目录
   cd 目录(可以是绝对路径)
注：文件及目录名称是区分大小写的
注：cd 后面的目录要求存在才能打开，否则报错
2、ls  展示当前目录下的文件或目录
3、touch  创建一个新的文件
   touch 文件名称
文件名.后缀名
后缀名任意   docx  txt   abc.data
4、mkdir  创建目录
5、rm  删除文件及目录
   rm 文件名   （只能删除文件）
   rm -rf 目录名 （删除目录及该目录下所有的内容）
6、rmdir  删除目录，只能删除空目录
7、man  进入命令的使用手册（英文版本）
   man  命令名称
8、help 进入命令的使用手册（英文版本）
   命令名称 --help
9、clear  命令  清屏
10、ctrl + L  快捷键  清屏
11、ctrl + C  强制退出


在线linux命令帮助手册（中文版）

十三、SSH
   SSH : 远程控制管理工具
   云服务器 ： IP地址 
   在linux下需要安装ssh才能使用： apt install ssh
   启动服务器： service ssh start
     
   ssh：默认端口号22
   win远程链接：Xshell
       输入Linux系统的IP地址，该地址下的账号及密码
   

十四、Linux下的用户系统
1、普通用户：自己创建的用户，平时尽量使用普通用户
2、超级管理员(root用户)：无敌的存在，可以摧毁系统中任意内容
3、当目前为普通用户，但又想执行root用户下才能执行的内容，使用sudo执行该命令。
4、!! : 代表上一次执行的命令

十五、Linux切换链接
Linux下允许同时存在七个登陆终端界面
ctrl+alt+f1-f6  : 黑终端窗口  tty1-tty6
ctrl+alt+f7     : 图形化界面  tty7
