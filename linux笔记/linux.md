# linux

## 常用命令

~~~em
cd 打开一个目录（目录可以是绝对路径）
	注：文件名和目录区分大小写。
	注：cd后面的目录要求存在，才能打开，否则报错
	
ls 展示当前目录下的文件或者目录

touch 创建一个新的文件
mkdir 创建目录

rm 删除文件及目录
rmdir 删除目录（只能删除文件）
rm -rf 删除目录及该目录下所有的内容

man 进入命令的使用手册（英文版）
man 命令名称

命令名称 --help    命令手册

在线linux命令帮助手册（中文版）
http：//man.linuxde.net

clear 清屏命令   Ctrl + L 快捷键清屏
Ctrl + C 强制退出

pwd：展示当前所在目录

mkdir：创建目录
	mkdir 目录名

ln：建立链接
	建立硬链接
	ln 源文件名  链接文件名

find : 查找文件
	格式：find[选项][参数]
	无选项及参数时，查找当前目录下所有文件和目录
	查找指定目录：find 目录名
	按指定名称查找：find 目录名 -name 条件
tar ： 打包解包  压缩与解压缩
	1.打包解包：
	打包：tar -cvf 打包后的名称 源文件（只进行打包，不压缩）
	解包：tar -xvf 文件名称/路径（.tar）      （将文件解包到当命令的文件目录下）
	解包：tar -xvf 文件名称/路径（.tar） -C 新的目录  （将文件解包到指定文件目录		 下）
	2.压缩与解压
	压缩：tar -zcvf 压缩文件的名称（以.gz为扩展名） 源文件
	解压缩：tar -zxvf 文件名称（.gz）  （将文件解包到当命令的文件目录下）
	解压缩：tar -zxvf 文件名称/路径（.gz） -C 新的目录  （将文件解包到指定文件目录下）
	
扩展名：
	扩展名对源文件无任何影响，建议打包的文件夹后缀名为.tar，在linux中压缩的扩展名建议写成.gz或.tgz格式。
	
file 查看文件类型

whereis：查看系统路径

ps：显示系统进程
ps -e ：显示所有进程
top:显示当前消耗资源最多的进程
kill：杀死进程
kill 进程号（PID）
kill -9 PID ：强制杀死进程

free：查看当前剩余空间

cal ： 日历

who：查看当前登录用户

route：查看当前路由

## linux系统运行状态

~~~
1.    0：关机状态
2.    1：单用户状态，root用户状态，linux系统维护时只有root有效时。
3.    2：多用户状态，没有网络文件系统
4.    3：完全多用户状态，工作中常见的状态之一（只有黑终端窗口）
5.    4：系统保留状态，未使用
6.    5：完全多用户状态，工作中常见的状态之一（有图形化界面，开发中常用的状态）
7.    6：关机并重启的状态

获取当前用户运行级别：
	runlevel
	who -r
	who      （获取当前登录的用户的信息）
切换运行级别：
	init 界别Number
	init 6
~~~

## 系统目录

~~~
1.进入根目录： cd/   进入系统的根目录
  /:根目录，当前电脑下所有的文件及目录都在根目录下。/就是linux系统的最高领导者
2.不同版本的linux系统的根目录有细微差别，单独去查看。
3.linux中的概念：将所有的东西，内容都当成文件来处理（键盘、鼠标、光盘等被抽象为一个   文件来处理）
4. bin:可执行的二进制文件，通常放置系统自带的二进制文件
5. dev:Device，设备通常将系统的硬件抽象成一个文件放在这里
6.initrd.img:系统最小的一个映像文件，包含着一个最小的Linux系统。
7.lost+found:异常关机时产生的错误日志或错误文件
8.opt:Options，存放开发者自己开发并安装的软件，属于可选文件
9.run:系统运行时必要的文件。
10.srv:系统服务启动时需要访问的文件或数据目录。
11.usr:存放与用户直接相关的文件目录。
12.boot:启动时需要的配置文件，核心文件
13.etc:配置文件（系统中所需要的各种配置文件，除非安装某些软件时需要更改系统的配置文件，否则不要更改其任何内容）
14.lib、lib64：系统内置的库文件，自己安装的软件的景泰链接，windows中的dll相似
15.media：媒体，存放系统的的外接设备的文件（U盘、dvd、鼠标）
16.proc：系统的文件，存放系统信息。
17.sbin：超级管理员特用的二进制文件（root用户）。
18.sys：系统的文件
19.var：存放系统中随时更改的文件。
20.cdrom：挂载入口，该目录在刚刚创建时一般为空，一般将光驱的文件挂在该目录下。
21.home：家，每个用户都会在home下拥有一个自己的目录，该目录名称与用户名一致
22.mnt:临时挂载入口。
23.root:超级管理员目录。
24.snap:全新的管理软件包的方式。
25.tmp:临时文件
26.vmlinuz:软链接，链接文件为boot目录下的一个vmlinuz..文件

~~~

## 用户系统

~~~
sudo命令:当前用户为普通用户，但让超级管理员去执行该命令
设置超级管理员密码：
sudo passwd root
passwd 修改用户密码
	1.默认只能修改当前用户的密码
	2.命令前有sudo，可以修改任意一个用户的密码
	3.在root用户下，可以修改任何用户的密码，
	4.用法：passwd[选项][用户名]
		当选项及用户名全部不写时，默认只能修改当前用户的密码
		当有用户名时，更改用户名的密码
		当选项为-d时，代表删除用户的密码
		想要进入linux系统用户必须存在密码，没有密码的用户类似无效
~~~

## 创建新的用户

~~~
1.创建新的用户
sudo useradd -m lucy
-m：会自动在home目录下创建一个与用户名一直的用户目录
2.创建的新用户需要设置密码
sudo passwd 用户名
3.查看用户所属群组
groups 用户名
例子：groups lucy
	==》lucy ： lucy ==》用户名 ： 主组
4.查看当前系统下存在的组：
文件为：/etc/group
5.创建组
sudo groupadd 组名
6.修改用户
group 用户名
例子：group lily
	==》
7.删除用户
sudo userdel 用户名（创建用户时自动生成的文件目录不会被删除，只是将用户删除了）
sudo userdel -r 用户名（创建用户时自动生成的文件目录会全部清理干净，）


	
~~~

## 文件系统

~~~
1.在不同的操作系统下，使用的文件格式是不同的
	windows：ntfs
	mac ： hfs
	linux ： ext2 ext3 ext4
fat32：在各个平台下均能使用，小于4G；大于4G；使用fat64
2.linux中的文件特性：
linux下所有的文件都是通过权限设置的。
	2.1权限分为十位
		第一位为类型
		-:文件
		d：目录
		l：链接
		后九位，每三位为一组
		第一组为当前用户的权限
		第二组为当前组的权限
		第三组为其他组的权限
3.修改文件权限
	3.1 修改文件权限命令
		chmod [ugoa][+-][rws]
		
~~~
